📋 Document Roadmap

Phase 1: Registration & Compliance (9 docs)

✔ Document Description
CIPC Registration Certificate Proof of company registration
Memorandum of Incorporation (MOI) Company constitution
SARS Registration (IT, PAYE, VAT) Tax compliance (VAT if turnover >R1m)
COIDA Registration Employee injury cover
Public Liability Insurance Cover for client site visits
Professional Indemnity Insurance For software errors, data loss – critical
Cyber Liability Insurance For data breaches, ransomware
B‑BBEE Certificate For corporate and government tenders
CSD Registration For government tenders

Phase 2: Contracts & Client Docs (12 docs)

✔ Document Description
Software Development Agreement (Master) For custom builds
IT Support / Maintenance Agreement (SLA) Ongoing services
Website Development Agreement Design, hosting, maintenance
Non‑Disclosure Agreement (NDA) Protect source code, client data
Data Processing Agreement (DPA) For POPIA compliance (client data)
Hosting / Server Colocation Agreement With data centre provider
Software Licence Agreement Terms of use for selling software
Quotation / Proposal Template Per project
Change Order / Variation Form Scope changes
Acceptance Testing Sign‑Off (UAT) Client confirms delivery
Warranty / Maintenance Period Addendum Bug fixes for 30–90 days
Subcontractor Agreement (freelance developer/designer) Outsourced work

Phase 3: Human Resources (10 docs)

✔ Document Description
Employment Contract – Software Developer Permanent staff
Employment Contract – IT Support Technician Help desk, onsite
Employment Contract – Project Manager Client delivery
Employment Contract – Sales / Account Manager Business development
Independent Contractor Agreement (freelance coder) Project basis
Code of Conduct (Employees) IP protection, professionalism
Disciplinary Code & Procedure HR compliance
Remote Work Policy Home office expectations
Attendance / Timesheet Register Client costing
Training Record (certifications – AWS, Azure, etc.) Skills tracking

Phase 4: Operations & Project Management (12 docs)

✔ Document Description
Project Brief / Requirements Document Client needs
Statement of Work (SOW) per project Detailed scope
Project Plan / Gantt Chart Timeline
Weekly Progress Report (client) Transparency
Bug / Issue Tracker Log Development and support
Change Request Log Scope changes
Timesheet Log (per employee / per project) Costing and billing
Code Repository Access Log (Git, etc.) Security
Backup & Disaster Recovery Log Data integrity
Server / Website Uptime Monitoring Log For SLA reporting
Client Login / Password Register (secured) Access management
Intellectual Property Assignment Form Client owns code after payment

Phase 5: Health & Safety (7 docs)

✔ Document Description
Occupational Health & Safety (OHS) Policy Office H&S
Ergonomic Workstation Assessment (per employee) Prevent RSI, back pain
Incident / Accident Report Form Office injury
First Aid Kit Register (office) Inventory
Fire Extinguisher Inspection Log (office) Monthly check
Emergency Evacuation Plan (office) Map and procedure
Remote Work Health & Safety Checklist For home office staff

Phase 6: Finance & Suppliers (8 docs)

✔ Document Description
Invoice Template (per project / retainer) Billing
Supplier Invoice Register (licences, hosting, hardware) Expense tracking
Payroll Register Staff salaries
Petty Cash Log Small office expenses
Retainer / Subscription Billing Schedule Monthly recurring clients
Client Payment History Accounts receivable
Supplier Payment Schedule Creditors aging
Bank Reconciliation Template Monthly finance

Phase 7: Equipment & Assets (9 docs)

✔ Document Description
Asset Register (Master) Computers, servers, monitors, phones
Software Licence Register (Microsoft, Adobe, etc.) Compliance
Domain Name Register (all client domains) Renewal tracking
SSL Certificate Renewal Log Expiry reminders
Equipment Maintenance Log (cleaning, hardware) Service records
Vehicle Log Sheet (if onsite support) Fuel, trips
Tool / Development Environment Access Log IDE, DB tools
Cloud Subscription Register (AWS, Azure, Google) Cost management
Asset Disposal Form Retirement of old hardware

Phase 8: Forms & Templates (8 docs)

✔ Document Description
Client Onboarding Checklist New client setup
Offboarding / Data Handover Checklist Project closure
Overtime Authorization Form Staff approval
Leave Application Form Staff requests
New Employee IT Access Request Form Email, repos, VPN
Supplier Evaluation Form (hosting, software vendor) Performance
Client Satisfaction Survey Feedback
Termination Checklist (offboarding) Return of equipment, revoke access
